Before Everything,
set up  new email account.
Primary Email acc is not recommended.
In your new email account,
go to manage account -> Security -> Less Secure App -> Turn on.

Now,
Inorder to automate the mail in the desired time daily.
you will have to create account in pythonanywhere.com--
Then goto files->upload all these files,
then goto Tasks -> Enter- python3 main.py -in Command Box --> Set up time in UTC(24h)->Check.
And you are good to go.
